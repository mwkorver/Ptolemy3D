pTolemy3D:

pTolemy3D is the open source 3D viewer.

Use the code area tab to test your own code. Use the different tabs to know the available options and learn to code.

Viewer Controls:

You can control pTolemy3D viewer using:
Keyboard:
  • a - zoom in
  • z - zoom out
  • s - zoom in
  • x - zoom out
  • d - rotate left
  • f - rotate right
  • r - tilt down
  • c - tilt up
  • left arrow - spin globe left
  • right arrow - spin globe right
  • up arrow - spin globe up
  • down arrow - spin globe down
Mouse:
  • drag - moves camera
  • Code Area
  • Camera
  • Terrain
  • Plugins
