Viewer Controls:

You can control pTolemy3D viewer using:
  • Keyboard:
    • a - zoom in
    • z - zoom out
    • s - zoom in
    • x - zoom out
    • d - rotate left
    • f - rotate right
    • r - tilt down
    • c - tilt up
    • left arrow - spin globe left
    • right arrow - spin globe right
    • up arrow - spin globe up
    • down arrow - spin globe down
  • Mouse:
    • drag - moves camera
