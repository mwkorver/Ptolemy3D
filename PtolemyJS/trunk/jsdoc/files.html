Matrix.js

This contains classes and functions related to Matrices. Matrices are stored in a 16 length array in row major order, because "visually" it is more easy to specify the values of the matrix at creation time. Independently of this 'toArray' method allows to return the matrix as an array in both column major order (ready for OpenGL operations) or row major order.
Author:
Antonio Santiago asantiagop(at)gmail(dot)com

MatrixStack.js

MatrixStack tries to emulate the matrix stack offered by OpenGL. MatrixStack allows to push and pop matrices (implemented as Matrix objects).

Globe.js

Globe contains the objects to be rendered. The Globe can have one or more elevation models which provides the terrain elevations. Elevation models are used from the beggining to the end of the array to server terrain elevation information. In the same way a Globe contains an array of RenderableObject which are rendered from the beggining to the end of the array.
